# 理财经理操作台前端

一个完整的理财经理操作台前端界面，提供客户管理、投资建议、数据分析等功能。

## 🚀 在线演示

**部署地址**: [https://z6jopx0q8g1d.space.minimaxi.com](https://z6jopx0q8g1d.space.minimaxi.com)

## 📋 功能特性

### 1. 客户管理仪表板
- 📊 客户关键指标概览
- 📈 数据可视化图表（客户风险分布、投资类型分布、客户资产分布）
- 🔍 客户搜索和筛选功能
- 📋 客户列表管理（支持网格和列表视图）
- 🎯 客户状态跟踪（活跃、待跟进、潜在、流失）

### 2. 客户详情页面
- 👤 完整的客户画像展示
- 💰 投资组合可视化分析
- 📊 收益趋势图表
- 🏷️ 风险分析和评估
- 💬 沟通记录管理
- 📋 投资明细展示

### 3. 投资建议管理
- 🤖 AI投资建议生成
- 📝 建议状态跟踪（待确认、已确认、已执行、已拒绝）
- 📊 建议统计分析
- 🔍 建议搜索和筛选
- 💬 客户沟通集成

### 4. 数据可视化
- 🥧 饼图：客户风险偏好分布、投资产品类型分布
- 📊 柱状图：投资产品分布、收益率对比、客户资产分布
- 📈 折线图：年度收益趋势
- 📱 响应式图表设计

## 🛠️ 技术栈

- **前端框架**: React 18.3 + TypeScript
- **构建工具**: Vite 6.0
- **样式框架**: Tailwind CSS 3.4
- **路由**: React Router v6
- **图表库**: Recharts 2.15
- **图标库**: Lucide React
- **包管理器**: pnpm

